# 숫자 문자열과 영단어

https://school.programmers.co.kr/learn/courses/30/lessons/81301

간단히 문자열을 모두 숫자로 교체

```js
function solution(s) {
    const NUMS = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"]
    
    for (let i = 0; i < 10; i++) {
        s = s.replaceAll(NUMS[i], i)
    }
    
    return Number(s)
}
```
